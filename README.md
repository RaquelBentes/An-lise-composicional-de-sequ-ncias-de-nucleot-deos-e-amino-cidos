# An-lise-composicional-de-sequ-ncias-de-nucleot-deos-e-amino-cidos
Esse projeto tem como objetivo analisar a composição de sequências de nucleotídeos e aminoácidos, realizando transcrição, tradução, geração do reverso complementar e cálculo do conteúdo gc, a fim de compreender características estruturais e funcionais das sequências biológicas.
